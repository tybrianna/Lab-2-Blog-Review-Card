## Overview

The goal for this challange was to replicate the blog preview card solution as close as possible. I was able to replicate it by adding a flex box to organize everything and apply rounded borders, background colors, and different text sizes to where it best fit. To have everything aligned properly I used padding and margins to align everything. I also made adjustments to the images sizes to make them fit properly. I also had to categorize some sections into individual classes to be able to customize them. This helped me to only make a change to one thing and not the whole section.

### What I learned

While trying to add rounded edges to my main image, I ran into issues doing to. After constantly playing around with it I realized that the units have to be the same for the paddings and borders. So if using pixels, the units must be in pixels for all, can't mix/match with percentages. This is the main thing that I learned that wil stick with me moving forward.
